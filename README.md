# Gaming Ecommerce Project 🎮🛒

## Overview
This is a **full-stack e-commerce application for gaming products**.
Backend is built with **ASP.NET Core**, and the frontend is built with **Angular**.

## Features
- User registration & authentication
- Product browsing & searching
- Shopping cart management
- Wishlist management
- Order processing
- Admin dashboard

## Tech Stack
- **Frontend:** Angular, TypeScript, HTML, CSS, Tailwind
- **Backend:** ASP.NET Core, C#
- **Database:** SQL Server / Entity Framework
- **Authentication:** ASP.NET Identity
- **Version Control:** Git & GitHub
